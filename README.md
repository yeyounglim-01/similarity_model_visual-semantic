상표권 문제를 위한

**시각적 특징 분석 + 개념적 의미 이해**를 결합한 하이브리드 이미지 검색 모델

## 🎯 개요

두 가지 상이한 접근 방식으로 이미지 유사도를 측정합니다:

| 모델 | 기술 | 특징 | 사용 사례 |
|------|------|------|---------|
| **시각적 모델** | ResNet50 Triplet Network | 색상, 형태, 레이아웃 기반 | 로고 매칭, 제품 이미지 검색 |
| **관념적 모델** | [설명: 의미론적 특징] | 이미지의 개념/의도 파악 | 카테고리 분류, 컨텍스트 검색 |
